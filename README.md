# 基米工艺 (Jimi Craft)

Minecraft 26.1.2 Fabric 模组 —— 让驯服的猫猫在结构中自动工作，为你挖矿、砍树、种猫薄荷。

## 结构

### 矿场
- **生成**：将铁镐丢在石头上
- **工作**：驯服的猫在结构范围内自动挖矿，坐卧均可
- **产出**：矿石自动存入结构内的箱子

### 伐木场
- **生成**：将铁斧丢在箱子上
- **工作**：驯服的猫在结构范围内自动砍树
- **产出**：主世界全部 10 种木材，均匀分布，自动存入箱子

### 猫薄荷工厂
- **生成**：将猫薄荷丢在羊毛上
- **激活条件**：结构内凑满 5 只驯服的猫
- **效果**：激活后 50 格半径内所有猫持续获得猫薄荷加成
- **提示**：工厂内的猫头顶显示「栽培猫薄荷中...」

## 猫薄荷

- **获取**：打草（短草丛/高草丛/蕨类）有 8% 概率掉落
- **使用**：右键已驯服的猫，消耗 1 个，猫获得 5 分钟加成
- **加成效果**：90% 产出概率 + 稀有率提升 + 50% 额外掉落
- **特效**：有加成的猫头顶冒爱心粒子

## 特性

- 无需 GUI、无需绑定物品 —— 结构即工作区
- 猫越多稀有物概率越高（最多 5 只加成）
- 猫工作时头顶显示产出物名 + 粒子特效
- 全主世界群系猫刷新率大幅提升
- 所有结构数据 JSON 持久化，重启服务器不丢失
- 中英双语支持

## 下载

[Releases](https://github.com/gggiz/jimicraft/releases) 页面下载最新版 JAR，放入 `mods` 文件夹即可。

## 构建

- **Minecraft**: 26.1.2 (Mojang mappings)
- **Java**: 25
- **Gradle**: 9.5.1
- **Fabric Loom**: 1.17.11
- **Fabric API**: 0.151.0+

```bash
./gradlew build
```

JAR 输出在 `build/libs/jimicraft-1.0.1.jar`。

## 自定义结构

模组自带了示例结构。如需自定义：

1. 在创造模式用结构方块保存你的建筑为对应名称的 `.nbt`
2. 替换 `src/main/resources/data/jimicraft/structure/` 下的对应文件：
   - `quarry_structure.nbt` — 矿场
   - `lumber_mill_structure.nbt` — 伐木场
   - `catnip_factory_structure.nbt` — 猫薄荷工厂
3. 确保结构内有箱子（矿石/木材会自动存入）
4. 重新构建模组
